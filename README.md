# 🚀 Personal Portfolio Website

A modern, responsive, and professional personal portfolio website built using **HTML, CSS, and JavaScript**. This project is designed to showcase skills, projects, experience, and contact information in a clean and elegant UI.

---

## 🌟 Features

- ⚡ Fully Responsive Design (Mobile, Tablet, Desktop)
- 🎨 Modern UI/UX with smooth animations
- 🌙 Dark / Light Mode Support (optional)
- 📁 Project Showcase Section
- 👨‍💻 About Me Section
- 🛠 Skills Section with progress indicators
- 📬 Contact Form (Email / WhatsApp integration optional)
- 🚀 SEO Optimized for Google indexing
- 🔥 Fast Loading & Lightweight code

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox / Grid / Animations)  
- JavaScript (Vanilla JS)
